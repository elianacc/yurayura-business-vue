# YuraYura-Business-Vue



## 简介

> YuraYura-Business-Vue为YuraYura后台管理的前端部分，为使用者提供一个良好交互的界面及相关的管理操作

## 安装

```
npm install
```

## 编译运行
```
npm run serve
```

## 打包
```
npm run build
```

## 项目构成

- ​    vue版本： vue2.x
- ​    脚手架： vue-cli 4.x
- ​    第三方UI框架： Element-UI 2.x  +  Bootstrap5.x
- ​    路由： vue-router
- ​    接口请求： axios
- ​    icon库： font-awesome

## 开发工具

- ​    开发环境： Node.js
- ​    编码： VS Code
- ​    页面访问及调试： Google Chrome + Vue.js devtools
- ​    markdown： Typora
- ​    截图： Snipaste
